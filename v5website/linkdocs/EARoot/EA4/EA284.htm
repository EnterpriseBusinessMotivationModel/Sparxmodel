This section will contain a list of "key questions" that senior leaders in an organization seek to have answered.  For each key question, a view of the EBMM will be extracted reflecting the elements that can be used to answer that question.  As an organization seeks to implement a metamodel, they should consider which of these key questions they would seek to answer, and include those views in their instantiation of the EBMM.<p/>
